# 🤖 Projeto Android

Este é um projeto desenvolvido com **HTML e CSS**, inspirado no universo do Android, com foco em apresentar informações de forma organizada e visualmente agradável.

## 🚀 Sobre o projeto

O site foi criado como parte dos estudos em desenvolvimento web, com o objetivo de aplicar na prática conceitos fundamentais de estruturação e estilização de páginas. Projetos como esse são comuns em cursos de front-end, onde o foco é consolidar o uso de HTML semântico e CSS moderno ([Daniel Borges X][1]).

A página apresenta conteúdos relacionados ao Android, utilizando um layout bem estruturado, tipografia personalizada e elementos visuais que deixam a navegação mais interessante.

Durante o desenvolvimento, foram trabalhados conceitos como:

* Estrutura semântica com HTML
* Estilização com CSS
* Uso de cores e identidade visual
* Responsividade básica
* Organização de conteúdo em seções

## 🎯 Objetivo

O principal objetivo deste projeto foi praticar e reforçar habilidades no desenvolvimento front-end, criando uma página completa, bem estruturada e com um design agradável, sem o uso de JavaScript.
